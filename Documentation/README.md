This directory is a loosely-organized collection of documentation.  

".md" files like this one are in [Markdown format](https://www.markdownguide.org/getting-started/). Most browsers will properly display these.

".pum" are the source files for (some) .png graphics diagrams. They're processed using the free [PlantUML](https://plantuml.com) tool, typically as Deployment diagrams.
